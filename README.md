# Wordle
Typescript wordle game: Full-Stack Dockerized

 This project features a React frontend using **Vite** and **Tailwind**, a **Node.js** backend, and a **PostgreSQL** database for word storage.

 Solution is kept securely in backend, rather than exposing to frontend.

# Start instructions

Ensure you have **Docker Desktop** installed, then run:

```
docker compose up --build
```
